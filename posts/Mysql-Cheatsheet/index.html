Mysql Cheatsheet

Jagdish Dev on Aug 132021-08-13T00:00:00+05:30
Updated Sep 192021-09-19T22:30:40+05:30 5 min read

MySQL Database

  • It is a Open source relational database management system.
  • Uses SQL
  • A leading database for web applications used for small apps to large enterprise apps
  • Cross Platform and used with multiple languages

Relational Database

  • Based on “relation model” of data
  • Virtually all RDBMS use SQL to manage them
  • Uses “tables” with “columns” and “rows”
  • Tables can be relate to eachother by keys

Comman Data Types

  • Numeric - INT, TINYINT, BIGINT, FLOAT
  • String - VARCHAR, TEXT, CHAR
  • Dates - DATE, DATETIME, TIMESTAMP
  • Others - BINARY, JSON

Installation & Environment

  • Local:

    • Terminal/GUI Install - Standalone Server
    • XAMPP, MAMP, WAMP (Usually used for local development)

  • Production:

    • Terminal installation via Linux package manager
    • Server software (CPanel, etc)

Docker containers/virtualization also an option

Management Tools

  • Terminal / Command line / Shell
  • Desktop Tools - MySQL Workbench, etc
  • Web Based Tools - PHPMyAdmin, etc

MySQL Locations

  • Mac /usr/local/mysql/bin
  • Windows /Program Files/MySQL/MySQL *version*/bin
  • Xampp /xampp/mysql/bin

1

CREATE TABLE posts(
id INT AUTO_INCREMENT,
   user_id INT,
   title VARCHAR(100),
   body TEXT,
   publish_date DATETIME DEFAULT CURRENT_TIMESTAMP,
   PRIMARY KEY(id),
   FOREIGN KEY (user_id) REFERENCES users(id)
);

INNER JOIN

  users.first_name,
  users.last_name,
  posts.title,
  posts.publish_date
FROM users
INNER JOIN posts

ORDER BY posts.title;

CREATE TABLE comments(
	id INT AUTO_INCREMENT,
    post_id INT,
    user_id INT,
    body TEXT,

);
